# 🌍 TripWay — Guia de Viagens

Projeto desenvolvido como parte do **Checkpoint 1**, com foco na criação de um site multipágina utilizando HTML e colaboração com GitHub.

---

## 📌 Objetivo

Desenvolver um site para uma empresa fictícia de viagens chamada **TripWay**, aplicando os seguintes conceitos:

- HTML semântico  
- Estrutura básica do HTML  
- Navegação entre páginas (multipage)  
- Caminhos relativos (relative path)  
- Formulários  
- Uso de imagens  
- Trabalho em equipe com Git e GitHub  

---

## 🧩 Estrutura do Projeto

O projeto é composto por 4 páginas principais:

### 🏠 Home (`index.html`)
- Apresentação da TripWay  
- Texto institucional  
- Imagem de destaque  
- Menu de navegação  
- Descrição da empresa  

---

### 🌎 Destinos (`pages/destinos.html`)
- Listagem de destinos turísticos  
- Cada destino contém:
  - Nome  
  - Descrição  
  - Imagem  

---

### 🧳 Serviços (`pages/pacotes.html` ou `servicos.html`)
- Lista de serviços oferecidos  
- Descrição detalhada de cada serviço  
- Imagens ilustrativas  

---

### 📩 Contato / Reserva (`pages/contato.html`)
- Formulário contendo:
  - Nome  
  - E-mail  
  - Destino de interesse  
  - Data  
  - Quantidade de pessoas  
  - Mensagem  
  - Botão de envio  

---
